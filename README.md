# Clashr-Auto-Desktop
---
### 无需学习，简单设置即可稳定使用！

![Clashr Auto Desktop](https://raw.githubusercontent.com/ClashrAuto/Clashr-Auto-Desktop/master/clashr.auto.png "Clashr Auto Desktop")

## 支持平台

| 平台    | 全局（支持游戏） | 普通（只代理网页） |
| ------- | ---------------- | ---------- |
| windows | 支持             | 支持       |
| MacOS   | 支持             | 不支持     |
| linux   | 不支持           | 不支持     |
